# Pokédex

<h2>Configuração do projeto</h2>

```
npm install
```

<h3>Compila aplicação - Inicia o servidor</h3>

```
npm run serve
```

<h3>Compila e minifica para produção</h3>

```
npm run build
```

<h3>Lints e corrige arquivos</h3>

```
npm run lint
```
